# **App Name**: Spot the Difference

## Core Features:

- Image Upload: Allow users to upload two images (JPEG/PNG) via file selection or drag-and-drop.
- Automated Comparison: Automatically detect and highlight differences between the two images using bounding boxes and color overlays, powered by AI. This feature will use an AI tool that selects the most relevant differences to display.
- Side-by-Side Display: Display the two images side-by-side with aligned views, showcasing the marked differences.
- Zoom Tool: Provide a zoom tool for users to inspect the detected differences more closely.

## Style Guidelines:

- Primary color: Gentle blue (#64B5F6) to create a calm, focused environment for detailed comparison work.
- Background color: Light gray (#F5F5F5), offering a neutral backdrop that minimizes distraction and maximizes visual clarity.
- Accent color: Soft orange (#FFAB40) to highlight detected differences and interactive elements, ensuring immediate user awareness and ease of interaction.
- Body and headline font: 'Inter' sans-serif for a clear, modern, and easily readable interface, enhancing user comprehension and comfort during detailed comparison tasks.
- Minimalist design to focus the user's attention on the image comparison task, reducing distractions and ensuring ease of use.
- Subtle transitions and highlights when differences are detected to guide the user's focus without overwhelming the visual field.
